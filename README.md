# Se quiser rodar o código localmente com npm
1- Abra o terminal e digite "git clone https://github.com/capellofabio/apl-calculator" no diretório onde deseja baixar.
2- Com o terminal no diretório do projeto, digite "npm run dev" e abra o link que aparecer.
